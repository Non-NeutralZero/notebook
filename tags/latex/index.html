    BibTex#

    Limit number of authors in IEEEtran#

    In the .bib file configure your IEEEtran as follows:

    @IEEEtranBSTCTL{IEEEexample:BSTcontrol,
CTLuse_forced_etal       = "yes",
CTLmax_names_forced_etal = "3",
CTLnames_show_etal       = "2" }
